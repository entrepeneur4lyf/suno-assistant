# App Icons

Place your app icons here. Required formats:

- `32x32.png` - 32x32 pixels
- `128x128.png` - 128x128 pixels
- `128x128@2x.png` - 256x256 pixels (Retina display)
- `icon.icns` - macOS icon (generate from PNG)
- `icon.ico` - Windows icon (generate from PNG)

You can use tools like:
- [Tauri Icon Generator](https://github.com/tauri-apps/tauricon)
- Online PNG to ICO/ICNS converters

For now, placeholder icons will be generated automatically by Tauri during first build.
